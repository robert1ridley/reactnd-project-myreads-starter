# MyReads Project

This is the first project from Udacity's React Nanodegree program. The app allows you to maintain a virtual bookshelf. You can also search books from the server and add the books you like to the shelf. 

## Starting up the project.

To get started developing right away:

* Ensure that you have npm or yarn installed on your local machine
* install all project dependencies with `npm install` or `yarn install`
* start the development server with `npm start` of `yarn start`
